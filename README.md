# Vikash - Frontend Developer Portfolio

A modern, responsive, and visually stunning personal portfolio website designed to showcase skills, projects, and education. Built with a focus on premium aesthetics, smooth animations, and user experience.

*   **Smooth Animations**: Intersection Observer-based fade-in animations, hover effects, and typing text effects.
*   **Interactive Elements**:
    *   Dynamic skill progress bars.

*   **Sections**:
    *   **Hero**: Introduction with a typing animation and call-to-action buttons.
    *   **About**: Personal introduction and profile image.
    *   **Skills**: Showcase of technical proficiency with circular progress indicators.
    *   **Education**: Timeline view of academic background.
    *   **Projects**: Filterable grid of featured projects.
    *   **Contact**: Functional contact form layout and social media links.
